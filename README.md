# Spellblog
Documentations, writings, spells and other stuff.

![vuepress-proof.png](https://imgur.com/xXpidn5)

*Run it locally:*

`npm install`

`npm run docs:dev`

or

`yarn install`

`yarn run docs:dev`

*To make changes/updates:*

Commit the changes to the main branch and the GitHub Pages will do its job.
